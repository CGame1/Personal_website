Research Interests

My research interests generally include:

  • Benthic ecology & biodiversity monitoring
  • Machine learning for environmental monitoring
  • Image processing and underwater image enhancement
  • Explainable and generalisable automated analysis workflows

Project focus

The current project develops ML methods that combine seabed images with environmental covariates (e.g., topography, temperature) to improve classification accuracy and ecological relevance. Emphasis is placed on methods that scale to large monitoring programs and provide interpretable outputs for stakeholders.
